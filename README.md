# DEHA – Vision-Based Real-Time Yoga Posture Visualization and Feedback System

## 📌 Overview
DEHA is a real-time yoga posture analysis system that uses computer vision to detect body landmarks, calculate joint angles, compare them with ideal reference postures, and provide color-coded feedback.

## 🎯 Key Features
- Live video capture using OpenCV
- Pose detection using MediaPipe
- Joint angle calculation
- Posture comparison with tolerance range
- Color-coded visual feedback:
  - 🟢 Green → Correct posture
  - 🔴 Red → Incorrect posture
- Continuous real-time frame processing

## 🛠 Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
